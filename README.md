# 🚀 Jota SNKS

> Projeto de e-commerce de tênis voltado ao desenvolvimento Full Stack, com foco na integração entre React no front-end e Java com Quarkus no back-end, aplicando boas práticas e explorando soluções modernas.

---

## 🧠 Sobre o Projeto

O **Jota SNKS** é um projeto criado para aprimorar habilidades em desenvolvimento web, simulando uma loja virtual moderna de tênis.

O principal objetivo é aprofundar o conhecimento na integração entre front-end e back-end, além de testar e evoluir habilidades em Java e React, explorando novas tecnologias e boas práticas de desenvolvimento.

---

## 💻 Tecnologias Utilizadas

### Front-End
- HTML5  
- CSS3  

### Futuro do Projeto
- React  
- Java + Quarkus
